# LoyaltyRewardApp
LoyaltyRewardApp là một ứng dụng bao gồm server và mobile app hỗ trợ việc cung cấp mã ưu đãi phía end user, và để shop cung cấp ưu đãi quảng bá shop của mình.
Ứng dụng được xây dựng bằng laravel, kotlin, compose với giao diện đơn giản, đẹp mắt. 

## Clone về máy
<code>git clone https://github.com/teamdptl/LoyaltyRewardApp</code>

Để chạy được app bạn cần phải cài đặt Laravel mới nhất và phần mềm android studio trong máy tính

## Tính năng
Sử dụng cho 2 phía:
- Endusers: Sử dụng mã khuyến mại của shop, tích điểm, sử dụng điểm.
- Shop: Cung cấp khuyến mãi, quảng cáo, đồng thời quản lý khách hàng quay lại sử dụng nhờ khuyến mãi nhằm đưa ra kế hoạch hợp lý.

- Trang Thanh Phúc, Võ Minh Tuấn (Frontend developer): Đảm nhiệm vai trò phát triển UI của ứng dụng bằng kotlin, compose

# LoyaltyRewardApp 
LoyaltyRewardApp is an application that includes a server and mobile app to support the provision of promotional codes to the end user, and for shops to provide incentives to promote their shops.
The application is built with laravel, kotlin, compose with a simple, beautiful interface.

## Backend Installation
<code>git clone https://github.com/teamdptl/LoyaltyRewardApp</code>

## Features
Use for 2 sides:
- Endusers: Use the shop's promotional code, accumulate points, use points.
- Shop: Provide promotions, advertising, and manage returning customers thanks to promotions to come up with a reasonable plan.

## Developers
- Huynh Khanh Duy, Nguyen Tuan Vu (Backend developer): Responsible for developing the backend of the application using Laravel, Firebase
- Trang Thanh Phuc, Vo Minh Tuan (Frontend developer): Responsible for developing the application's UI using kotlin, compose
